# CostesLab Test Data

Small datasets for validating CostesLab image analysis pipelines.
See [CostesLab](https://github.com/svcostes/CostesLab) for the analysis code.

## SpotDetection/human_blood_53bp1/
- 12 TIF images, 2-channel (DAPI + 53BP1), time course after irradiation
- T_0min through T_240min, 2 replicates each
- Used to validate foci_detection_v2.ipynb
- Pixel size: 0.1625 um/px
